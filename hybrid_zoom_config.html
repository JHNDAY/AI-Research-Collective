In the room
In-person attendees
Not on Zoom. Watch the presentation on the room display mirrored from Computer 1. Are visible to remote attendees via Computer 2's camera.
On Zoom
Remote attendees
Receive two feeds via Zoom: the shared screen from Computer 1, and the room camera view from Computer 2. Can pin or spotlight either feed in their own Zoom layout.
⚠ Critical — Audio
Prevent echo feedback
🔇
Mute mic on Computer 2. Two live mics in the same room will create a feedback loop.
🔈
Disable speaker on Computer 2. Only one device should play audio in the room.
🎤
Use Computer 1 (or an external room mic) as the single audio source for Zoom.
Setup tips
Get the best result
📷
Wide-angle lens on Computer 2 helps capture the whole in-room group.
📌
Spotlight Computer 2 for remote attendees so the room view stays visible.
🖥
Position Computer 2 so the camera faces in-room seating, not the presenter's back.
