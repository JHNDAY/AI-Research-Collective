Finding Your Keys

EASY
Learn to identify and locate the special keyboard modifier keys

Understanding Mac Key Symbols

Before using keyboard shortcuts, you need to know where these four special keys live on your Mac keyboard.

SymbolKey NameWhere to Find It
⌘CommandBottom row, immediately left and right of the spacebar
⌥OptionNext to the Command keys; says "option" or "alt"
⇧ShiftLeft and right sides, above Command — has an upward arrow
⌃ControlBottom left corner; says "control" or shows "^"
💡 Memory Tip: Most shortcuts use Command (⌘) as the main key. Think of it as the "Mac action key" — it's your new best friend!
Practice: Key Location Drill
  1. Place your left index finger on the Command key (⌘)
  2. Move it to the Option key (⌥)
  3. Find the Shift key (⇧) with your left pinky
  4. Locate the Control key (⌃)
  5. Repeat 5 times until you can find each key without looking
Checkpoint: Can you do this?
  • I can find the Command key (⌘) without looking
  • I can find the Option key (⌥) quickly
  • I know where both Shift keys (⇧) are
  • I can locate the Control key (⌃)

Understanding Windows Keyboard Modifier Keys

Before using shortcuts, you need to know where the special modifier keys are on your Windows keyboard.

🪟 Windows Keyboard Keys

  • Ctrl (Control) — Bottom left and right corners
  • Alt — Next to the spacebar (both sides)
  • Shift — Large keys at far left and right
  • ⊞ Windows key — Between Ctrl and Alt (left side)
💡 Key Translation from Mac: ⌘ Command (Mac) = Ctrl (Windows)  |  ⌥ Option (Mac) = Alt (Windows)  |  Shift is the same on both!
Checkpoint: Can you do this?
  • I can find both Ctrl keys without looking
  • I can find the Alt key quickly
  • I know where both Shift keys are
  • I can locate the Windows (⊞) key

Opening & Managing Tabs

EASY
Master tab creation, switching, and closing with keyboard shortcuts

Tabs are the foundation of modern web browsing. During BOLLI Zoom sessions, you'll often need multiple websites open at once!

🧭 Safari Tab Shortcuts

  • ⌘+T — Open new tab
  • ⌘+W — Close current tab
  • ⌘+1–9 — Jump to tab by number
  • ⌘+Z — Reopen last closed tab
Practice Exercise: Opening Multiple Tabs
  1. Press ⌘+T to open a new tab — practice this 4 times
  2. Navigate to different sites: Tab 1 = Brandeis, Tab 2 = NY Times, Tab 3 = Guardian, Tab 4 = Google
  3. Practice jumping: Press ⌘+1 (Brandeis), ⌘+3 (Guardian), ⌘+2 (NY Times)
  4. Close a tab with ⌘+W, then reopen it with ⌘+Z
⚠️ Common Mistake: Hold Command BEFORE you press the letter or number. Don't try to press them at exactly the same time!

🍎 Mac Chrome

  • ⌘+T — New tab
  • ⌘+W — Close tab
  • ⌘+1–9 — Jump to tab
  • ⌘+⇧+T — Reopen closed tab

🪟 Windows Chrome

  • Ctrl+Shift+T — Reopen closed tab
💡 Chrome Split View (Windows): Right-click any tab and choose "Add tab to new split view" — Chrome splits the browser window in two instantly!

🍎 Mac Firefox

  • ⌘+⇧+T — Reopen closed tab

🪟 Windows Firefox

💡 Firefox Tab Groups: Right-click any tab and select "Add Tab to Group" to organize tabs into color-coded groups — very handy when juggling multiple BOLLI courses!

🪟 Windows Edge

💡 Edge Vertical Tabs: Edge offers a unique vertical tab bar on the left side — great for managing many tabs! Click the tab icon at the top-left of the browser to enable it.

Creating & Managing Bookmarks

EASY
Save important websites and organize them for quick access to course materials

Bookmarks give you quick access to your Zoom links, course materials, and BOLLI homepage — no more retyping URLs!

🧭 Safari Bookmark Shortcuts

  • ⌘+D — Bookmark current page
  • ⌘+⌥+B — Open/hide Bookmarks sidebar

📁 Creating a Folder in Safari

  1. Press ⌘+⌥+B to open the Bookmarks sidebar
  2. Right-click anywhere in the sidebar
  3. Choose New Folder
  4. Type a name (e.g. "BOLLI Courses") and press Return
  5. Drag bookmarks into the folder to organize them
Practice: Bookmark and Organize
  1. Go to www.brandeis.edu, then press ⌘+D
  2. Edit the name to "BOLLI Home" and click Add
  3. Press ⌘+⌥+B to open the Bookmarks sidebar
  4. Right-click in the sidebar and choose "New Folder" — name it "BOLLI Courses"
  5. Bookmark zoom.us and theguardian.com/us, saving each into your "BOLLI Courses" folder
💡 Naming Tip: Give bookmarks short, clear names — "BOLLI Home" is much better than "Brandeis University Official Website Homepage"!

🍎 Mac Chrome

  • ⌘+D — Bookmark page
  • ⌘+⇧+O — Open Bookmark Manager
  • ⌘+⇧+B — Show/hide Bookmarks Bar

🪟 Windows Chrome

  • Ctrl+D — Bookmark page
  • Ctrl+Shift+O — Open Bookmark Manager
  • Ctrl+Shift+B — Show/hide Bookmarks Bar
Practice: Bookmark and Organize
  1. Go to www.brandeis.edu and press ⌘+DCtrl+D
  2. Rename it "BOLLI Home" and click Done
  3. Open the Bookmark Manager and create a "BOLLI Courses" folder
  4. Toggle the Bookmarks Bar on for quick access

📁 Creating a Folder in Chrome

  1. Press ⌘+⇧+O to open Bookmark Manager
  2. Press Ctrl+Shift+O to open Bookmark Manager
  3. Right-click on any existing folder in the left panel
  4. Choose Add new folder from the menu that appears
  5. Type a name (e.g. "BOLLI Courses") and press Enter
  6. Drag bookmarks into the new folder to organize them
💡 Shortcut: You can also right-click directly on the Bookmarks Bar (the strip under the address bar) and choose Add folder — no need to open the Manager at all!

🍎 Mac Firefox

  • ⌘+D — Bookmark page
  • ⌘+⇧+O — Open Library (bookmarks)
  • ⌘+⇧+B — Show/hide Bookmarks Toolbar

🪟 Windows Firefox

  • Ctrl+D — Bookmark page
  • Ctrl+Shift+O — Open Library
  • Ctrl+Shift+B — Show/hide Bookmarks Toolbar

📁 Creating a Folder in Firefox

  1. Press ⌘+⇧+O to open the Library
  2. Press Ctrl+Shift+O to open the Library
  3. Right-click on Bookmarks Menu in the left panel
  4. Choose New Folder
  5. Type a name and click Add
  6. Drag bookmarks into the folder to organize them

🍎 Mac Edge

  • ⌘+D — Add to Favorites
  • ⌘+⇧+O — Open Favorites Manager
  • ⌘+⇧+B — Show/hide Favorites Bar

🪟 Windows Edge

  • Ctrl+D — Add to Favorites
  • Ctrl+Shift+O — Open Favorites Manager
  • Ctrl+Shift+B — Show/hide Favorites Bar

📁 Creating a Folder in Edge

  1. Press ⌘+⇧+O to open Favorites Manager
  2. Press Ctrl+Shift+O to open Favorites Manager
  3. Click Add folder in the top right
  4. Type a name and press Return
  5. Drag favorites into the folder to organize them
💡 Edge Collections: Edge has a unique "Collections" feature (click the ✚ icon in the toolbar) — great for gathering research from multiple websites into named groups!
🎯 Organizing Tips: Create folders for BOLLI Courses, Current Events, Research Tools, and Zoom Links. Rename bookmarks to short clear names. Use the Bookmarks Bar for your most-visited sites!
Checkpoint: Bookmark Mastery
  • I can bookmark a page in 2 seconds

Taking Screenshots

MEDIUM
Capture images of your screen to save important information from class

Screenshots let you capture what's on your screen — perfect for saving course materials, error messages, or sharing info with others.

ℹ️ Note: Screenshot shortcuts are an operating system feature — they work the same regardless of which browser you use on your Mac!

🍎 Mac Screenshot Shortcuts

1 Entire Screen: ⇧+⌘+3 → Captures everything instantly; saves to Desktop
2 Selected Area: ⇧+⌘+4 → Cursor becomes a crosshair — click and drag to select the area you want
3 Specific Window: ⇧+⌘+4 then Spacebar → Cursor becomes a camera — click any window to capture it
4 Screenshot Toolbar: ⇧+⌘+5 → All screenshot options plus screen recording!
💡 Where do screenshots go? By default, they save to your Desktop as PNG files with timestamps. You can change this in the screenshot toolbar (⇧⌘5) options.
ℹ️ Note: Screenshot shortcuts are a Windows operating system feature — they work the same regardless of which browser you use!

🪟 Windows Screenshot Methods

1 Snipping Tool (Best method): ⊞ Win+Shift+S → Screen dims; choose rectangular, freeform, window, or full screen capture. Goes to clipboard — paste it anywhere!
2 Save Directly to File: ⊞ Win+PrtScn → Screen flashes; screenshot saves automatically to Pictures > Screenshots folder
3 Active Window Only: Alt+PrtScn → Captures only the currently active window to clipboard
💡 Best Method: Use ⊞ Win + Shift + S — it's the most flexible and user-friendly option in Windows!
Practice Exercise: Take a Test Screenshot
  1. Open a website — try www.brandeis.edu/osher
  2. Take a screenshot using your platform's method above
  3. Find your screenshot: check your Desktopif you used Win+Shift+S, paste it into a document; if Win+PrtScn, check Pictures > Screenshots
🎯 BOLLI Use Cases: Save a slide from a presentation • Capture discussion questions from Zoom chat • Save an error message to share with tech support • Capture course schedule information
Checkpoint: Screenshot Skills
  • I can capture my entire screen quickly
  • I can capture just a selected area of my screen
  • I know where my screenshots are saved
  • I've successfully taken and found a test screenshot

Split Screen & Window Management

MEDIUM
View multiple windows side-by-side for Zoom classes and efficient multitasking

During BOLLI Zoom sessions, you often need to see multiple things at once: the Zoom window, course materials, and your notes. Split screen makes this easy!

ℹ️ Note: On Mac, split screen is an operating system feature — it works the same in Safari, Chrome, Firefox, and Edge!

🍎 Mac Split Screen

1 Find the green button — top-left corner of any window (the red/yellow/green dots)
2 Click and HOLD the green button — hold for a moment until options appear (don't just click!)
3 Choose a side — select "Tile Window to Left of Screen" or "to Right of Screen"
4 Select the second window — click another open window to fill the other side
5 Adjust the divider — drag the center line to resize each side to your preference
6 Exit split screen — press Esc, or click the green button again, or press ⌃+⌘+F
💡 Quick Method: Hold ⌥ Option and click the green button to instantly tile the window to one side!
⚠️ Troubleshooting:
  • Green button does nothing: Try clicking and holding longer (1–2 seconds)
  • Windows won't tile: Make sure you're not in full screen mode first (press Esc)
  • Split looks wrong: Adjust the divider by dragging the center line

🪟 Windows Snap (All Browsers)

1 Keyboard Method (Fastest): Make a window active, then press ⊞ Win+← to snap it left, or ⊞ Win+→ to snap it right
2 Mouse Method: Click and drag a window's title bar to the left or right edge of the screen; release when the outline appears
3 Select the second window — Windows will show your other open windows; click one to fill the other half
4 Adjust the divider — drag the center line to resize
💡 Quarter Screen Snap: Win+Up = top half | Win+Down = bottom half | Win+Left then Up = top-left quarter
💡 Chrome Built-in Split View: Right-click any Chrome tab and choose "Add tab to new split view" — Chrome splits the browser window in two with no resizing needed!
💡 Edge Split Screen: Edge has a built-in split screen button in the toolbar (look for the two-rectangle icon). Click it to split your browser into two browsing panels side by side!
🎯 Recommended BOLLI Setup:
Discussion Class: Left = Zoom (see classmates) | Right = Course readings or research browser
Lecture Class: Left = Zoom (see presenter) | Right = Notes app or Google Doc
Current Events: Left = Zoom | Right = Browser with multiple news tabs
Checkpoint: Split Screen Mastery
  • I know how to activate split screen on my platform
  • I can create a split screen in under 10 seconds
  • I can adjust the split to my preferred layout
  • I know how to exit split screen
  • I can set up Zoom on one side and my browser on the other

Essential Navigation Shortcuts

EASY
Navigate websites quickly: address bar, back/forward, refresh, find text, zoom

🧭 Safari Navigation Shortcuts

  • ⌘+L — Jump to address bar
  • ⌘+[ — Go back  |  ⌘+] — Go forward
  • ⌘+R — Refresh page
  • ⌘+F — Find text on page
  • ⌘++ / - — Zoom in/out  |  ⌘+0 — Reset zoom
Practice: Quick URL Access and Find
  1. Press ⌘+L — address bar highlights. Type: nytimes.com and press Return
  2. Visit 4 more sites, then press ⌘+[ to go back through them
  3. Press ⌘+F, type "climate" — Safari highlights all matches
  4. Press ⌘++ three times to zoom in, then ⌘+0 to reset

🍎 Mac Chrome

  • ⌘+L — Address bar

  • ⌘+R — Refresh
  • ⌘+F — Find on page

🪟 Windows Chrome

Copy, Paste, and Text Management

EASY
Master essential text editing shortcuts that work in every application

These shortcuts work in your browser, Word, email, and almost every application — they are fundamental computer skills!

🍎 Mac Text Shortcuts (All Browsers)

  • ⌘+C — Copy selected text or content
  • ⌘+V — Paste
  • ⌘+X — Cut (copy and delete)
  • ⌘+Z — Undo  |  ⌘+⇧+Z — Redo
  • ⌘+A — Select all

🪟 Windows Text Shortcuts (All Browsers)

  • Ctrl+C — Copy selected text or content
  • Ctrl+V — Paste
  • Ctrl+X — Cut (copy and delete)
  • Ctrl+Z — Undo  |  Ctrl+Y — Redo
  • Ctrl+A — Select all

Text Selection Techniques

  • Double-click any word to select it
  • Triple-click anywhere in a paragraph to select the whole paragraph
  • Click and drag across text to select a custom range
  • ⌘Ctrl+A to select everything on the page
Practice Exercise: BOLLI Note-Taking Workflow
  1. Open your browser and go to any article on wikipedia.org
  2. Double-click a word to select it, then copy it: ⌘+CCtrl+C
  3. Open the Notes app or a Google Doc
  4. Paste with ⌘+VCtrl+V — the text appears!
  5. Repeat with a full paragraph: click and drag to select, copy, paste to your notes
🎯 BOLLI Note-Taking Workflow: During a Zoom class, find a relevant article in your browser → select the key quote → copy it → switch to your notes → paste it. All without typing it by hand!
Checkpoint: Text Management
  • I can select text by double-clicking, triple-clicking, and dragging
  • I can copy and paste text between windows
  • I can undo mistakes quickly
  • I've copied text from a webpage into a notes document
