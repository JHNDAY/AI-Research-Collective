Prompt Interpretation
Claude parses intent and content type from your message. Visual signals ("show me," "build," "visualize") are weighted alongside implicit need — data that benefits from structure, charts, or interaction.
intent detection content classification signal weighting
2
📐
Skill File Consultation
Before writing a single line, Claude reads the relevant SKILL.md — a curated guide encoding environment constraints, available libraries, design tokens, and rendering quirks. This step is mandatory and shapes everything that follows.
frontend-design docx / pptx / pdf environment constraints
3
🔀
Routing Decision
Two distinct rendering paths based on output type. Claude selects based on whether the deliverable is inline and ephemeral or persistent and deployable.
↳ two paths
💬 Inline Artifact

Uses show_widget tool. HTML/SVG renders in a sandboxed iframe directly in the chat stream. Ideal for charts, diagrams, interactive tools, games.

📁 File Output

Uses create_file + present_files tools. Writes a standalone .html to disk. Ideal for GitHub Pages, portals, downloadable deliverables.

4
🏗️
Code Architecture Planning
Claude designs the HTML mentally before writing — laying out semantic structure, CSS custom properties for theming, JavaScript logic, and data embedding strategy. Single-file constraint means CSS and JS live inside the HTML.
HTML structure CSS variables JS event model data strategy
5
⚙️
Code Generation
The full HTML is written using available libraries from cdnjs.cloudflare.com. Environment rules are strict: no localStorage, Tailwind utility classes only (no compiler), React hooks must be imported explicitly.
cdnjs only no localStorage single file

6
🔄
Iteration Loop
The conversation is a refinement cycle. You see the render, request changes ("darker palette," "add a third column"), and Claude edits surgically via str_replace — or rewrites entirely. Each change produces a new render in the stream.
str_replace edits full rewrites conversational refinement
7
🤖
Claude-in-Claude (Optional)
For AI-powered artifacts, the rendered HTML itself calls api.anthropic.com/v1/messages at runtime. The outer Claude builds the shell; the inner Claude responds to user actions inside the artifact. A live AI application, not static HTML.
live API calls runtime intelligence stateful conversation
Key Constraint Surface

All HTML artifacts run in a sandboxed iframe with no access to browser storage APIs. External scripts must load from cdnjs.cloudflare.com. Three.js CapsuleGeometry, localStorage, and sessionStorage are all off-limits. React artifacts use Tailwind's pre-compiled base stylesheet — no JIT, no custom classes.
